Making of the Game!

If you want to create a game in html, you would need to create a canvas like shown in line 25 of the code above.
In order for that canvas to have color and a border, you would need a id for the canvas.
It is recommended to use an id, since you will need to connect your canvas to javascript by using the code "getElementById"

Next thing what i did is customized the canvas. I gave it a white border line to start with
There is always going to be a moment where you try to do something and it doesn't always work out.
In order to make a background image for my canvas, first I tried to place a image on the html page (not canvas)
and position it so it looked like it was in the canvas. That didn't really work out, either canvas moved or the
image moved each time i refreshed my page. Then i used css to place the image in canvas as shown above.

The most crucial part and the most easy to fail part is javascript, this takes a lot of focus.
What i haved done there is not one day's work, it took me a week to learn the basics of javascript. I had to look
online for things i had to do. The loading function in the code above works with the loading bar and the video that pops up
when someone clicks the play button. It uses a lot of id's that is why it is recommended to use id's instead of classes
including the object to javascript. The video was fairly easy to put in, but doesn't always load-in.
